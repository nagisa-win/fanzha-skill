# 风险模式识别规则

本文件定义反诈守护助手的三级风险识别体系，供 SKILL.md 主逻辑引用。

---

## 一、AI 幻觉信号（Hallucination Signals）

AI 幻觉是指模型输出了不准确、无法验证或凭空捏造的内容，用户可能因此做出错误决策。

### H1 - 数据可信度风险
| 信号特征 | 示例 | 风险级别 |
|---------|------|---------|
| 具体数字无来源 | "该药物治愈率为 87.3%" | 🟡 L1 |
| 引用了不存在的文件/报告 | "根据 2025 年工信部 XX 号文件" | 🟠 L2 |
| 提供了无法验证的电话号码 | "官方客服电话：400-xxx-xxxx" | 🟠 L2 |
| 保证某事情"一定"发生 | "这个投资一定会涨" | 🔴 L3 |
| 医疗诊断/处方建议 | "你的症状是XX，应该服用XX药" | 🔴 L3 |
| 法律结论性判断 | "按照法律，你可以直接要求赔偿10万" | 🔴 L3 |

### H2 - 内容矛盾信号
| 信号特征 | 检测方式 | 风险级别 |
|---------|---------|---------|
| 同话题前后描述不一致 | 对比同一实体的多次提及 | 🟡 L1 |
| 与用户提供的原始信息矛盾 | 比对用户输入 vs 模型输出 | 🟠 L2 |
| 与常识性事实严重冲突 | 如声称某人去世时间错误 | 🟠 L2 |

### H3 - 过度自信信号
```
高风险词汇列表（出现即标注）：
- 保证 / 确保 / 100% / 绝对 / 一定 / 必然
- 无风险 / 零风险 / 稳赚 / 不会亏损
- 我可以证明 / 已经证实 / 科学证明
```

## 二、诈骗类型关键词矩阵

### 2.1 电信诈骗类 🔴 L3

```
公检法类：
公安局 / 检察院 / 法院 / 警察 / 逮捕令 / 通缉令 / 洗钱
资金审查 / 配合调查 / 保密账户 / 安全账户

冒充客服类：
退款 / 赔偿 / 理赔 / 刷单 / 返利 / 好评返现
账号异常 / 开通会员 / 解除风控 / 冻结账户

情感诈骗类：
网恋 / 杀猪盘 / 数字货币 / 虚拟货币 / 带你赚钱
感情投资 / 境外平台 / 专属渠道

中奖类：
中奖 / 恭喜获奖 / 抽到大奖 / 需缴纳手续费 / 领奖金
```

### 2.2 网络投资诈骗类 🔴 L3

```
高收益承诺：
月收益30% / 日化收益 / 稳定高息 / 躺赚 / 睡后收入
保本保息 / 内部渠道 / 私募基金（无牌照）

平台特征：
海外交易所 / 无监管平台 / APP私下下载 / 扫码充值
USDT / 数字钱包 / 提现需缴税

操控手法：
先小额盈利 / 冻结提款 / 追加保证金 / 亏损后补仓
```

### 2.3 钓鱼欺诈类 🟠 L2

```
链接特征：
- 域名拼写错误（如 b4idu.com、a1ipay.com）
- 使用 IP 地址访问
- 短链无法预览目标地址
- HTTP 非加密页面要求输入密码

内容特征：
- 要求填写身份证号 + 银行卡 + 验证码组合
- 声称"系统升级，需重新验证"
- "点击领取" / "扫码激活" / "限时免费"
```

### 2.4 假冒权威类 🟠 L2

```
冒充机构：
国家机关 / 银行 / 知名企业官方
权威媒体 / 知名医院 / 著名大学

冒充名人：
明星推荐 / 专家背书 / 名人代言（无法核实）
```

## 三、用户情绪与行为风险信号

情绪信号本身不直接定级，需结合内容综合判断：

| 情绪/行为信号 | 含义 | 关注点 |
|------------|------|-------|
| "不敢告诉家人" | 被要求保密 | 结合金钱类内容升至 L3 |
| "很急，今天必须" | 制造紧迫感 | 经典诈骗手法 |
| "对方说是警察/法院" | 冒充公检法 | 直接 L3 |
| "对方让我下载APP" | 远程控制风险 | 结合金钱升至 L3 |
| "已经转了一部分" | 可能已受骗 | 立即 L3 + 报案引导 |
| "对方说帮我投资" | 投资诈骗信号 | 🟠 L2，追问细节 |
| "感觉不太对劲" | 用户本能质疑 | 🟠 L2，引导验证 |

## 四、网络内容可信度评估标准

当工具抓取了外部网页/文章内容时，使用以下标准评估：

### 可信度高（无需预警）
- HTTPS 正规域名（与内容匹配）
- 有明确作者、发布机构、发布时间
- 内容平和、逻辑清晰、无催促语言
- 政府官网 (.gov.cn)、权威媒体、学术机构

### 可信度中等（🟡 L1 附注）
- 内容来自个人博客/论坛（非权威来源）
- 信息较旧（超过1年），可能已过时
- 无法找到原始来源

### 可信度低（🟠 L2 警告）
- 域名与声称机构不符
- 页面充满广告、弹窗
- 语言夸大、充满绝对化表述
- 无法找到机构真实注册信息

### 高危网页（🔴 L3 阻断）
- 要求输入银行卡/密码/验证码
- 检测到已知钓鱼域名特征
- 声称是"官方"但域名明显伪造
- 要求下载 APK 或打开摄像头/麦克风

## 五、组合风险升级规则

单一信号可能不触发高风险，但组合出现时需升级：

```
组合一：金钱 + 紧迫 + 保密 → 强制 L3
  例："现在必须转账，不能告诉家人，公安在查你"

组合二：陌生人 + 感情 + 投资 → 强制 L3（杀猪盘）
  例："认识不久的网友让我在他介绍的平台上投资"

组合三：AI输出 + 不可验证数字 + 金融建议 → L2
  例：模型给出了具体的股票代码和"保证收益"

组合四：链接 + HTTP + 要求登录 → L2
  例：陌生人发来的链接要求输入账号密码

组合五：已转账 + 任意诈骗信号 → 强制 L3 + 报案引导
```
