# 紧急求助资源

本文件列出反诈守护助手在 L3 高危响应时使用的求助资源，按场景分类。

---

## 一、核心报警与举报热线

| 热线 | 用途 | 何时使用 |
|------|------|---------|
| **110** | 报警 | 人身安全受威胁 / 已遭受诈骗 / 任何紧急情况 |
| **96110** | 国家反诈举报热线 | 举报诈骗 / 咨询是否受骗 / 预警拦截 |
| **12315** | 消费者投诉举报 | 网购诈骗 / 假冒伪劣 / 虚假广告 |
| **12345** | 政务服务热线 | 查询政府机构官方电话 / 非紧急投诉 |
| **120** | 急救 | 因诈骗导致情绪崩溃、身体不适 |

## 二、银行紧急止损

当用户已经转账或可能即将转账时，优先引导：

### 立即行动（黄金4小时）
```
第一步：挂断诈骗电话
第二步：立即拨打本人银行客服，申请"紧急止付"
第三步：拨打 110 报警，提供转账记录
第四步：拨打 96110 举报

越快报案，资金追回可能性越大！
```

## 五、核实机构真实联系方式的方法

引导用户通过以下途径核实，而非相信陌生人给的电话：

1. **官方网站**：直接在浏览器输入机构官网域名（勿点击陌生链接）
2. **政府机构**：通过 12345 或机构官网查询
3. **银行客服**：卡背面印有客服电话
4. **电商平台**：通过平台内官方渠道联系，不使用陌生人提供的联系方式
5. **企业查询**：通过"企查查"、"天眼查"等官方工具核实企业注册信息
