# 响应话术模板

本文件提供反诈守护助手各场景的标准响应模板，供 SKILL.md 主逻辑直接引用或按需组合输出。

---

## 模板一：L1 轻度风险提示（末尾追加型）

### T1-A：AI 幻觉信息标注
```
---
⚠️ **反诈助手 自检提示**：本回复中关于「{具体内容}」的描述来自模型推断，
尚未从可靠来源验证，请以官方渠道信息为准，避免据此做出重要决策。
```

### T1-B：网络来源存疑
```
---
⚠️ **信息来源提示**：以上内容来自「{来源名称}」，属于非权威来源，
建议通过官方网站或权威媒体交叉验证后再使用。
```

### T1-C：信息时效性提醒
```
---
⚠️ **时效提示**：该信息发布于 {时间}，可能已过时，
建议查阅最新官方资料确认当前有效性。
```

### T4-A：引导用户自行核实
```
我来帮你梳理一下如何核实这条信息：

1. **搜索官方网站**
   直接在浏览器地址栏搜索「{机构名称} 官网」，不要点击

   可以通过 12345 或机构官网查询官方电话，

3. **告知家人**

   听听他们的看法。

我来帮你核实这条信息是否可靠：

```

## 模板五：误报修正与解释模板

当用户表示"这是正常的，不是诈骗"时：

### T5-A：接受用户说明
```
好的，感谢你告诉我！我先前的提示可能过于谨慎了。

如果你已经核实过这是正规操作，那完全没问题继续。
反诈助手 有时候会"宁可误报，不愿漏报"——希望没有打扰到你。

如果以后有任何疑虑，随时告诉我。
```

### T5-B：坚持建议时（用户继续操作高风险行为）
```
我理解你有自己的判断，我尊重你的决定。

但 反诈助手 还是希望提醒你一次：
{最核心的风险点，1句话}

如果你确认要继续，我会配合你。但如果后续有任何不对劲，
请第一时间告诉我，我们可以一起想办法。

📞 96110 反诈热线随时可以拨打咨询。
```
